# LDashboard 仪表盘布局

常见中后台系统中的顶部栏和左侧栏固定的仪表盘布局。

滚动区域设置在了主内容区，有最小宽度，可以进行横向滚动。

## API

### Slots

#### (default)

主内容区。

#### head

设置顶部内容。

#### side

设置左侧栏内容。

#### main

重置中间区域内容。包裹在主内容区外。
