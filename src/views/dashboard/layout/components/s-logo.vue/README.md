# ULogo 标识

## 示例
### 基本用法

``` html
<u-linear-layout direction="vertical">
    <div style="background: #4289db;">
        <l-logo></l-logo>
    </div>
</u-linear-layout>
```

### 子标题

``` html
<u-linear-layout direction="vertical">
    <div style="background: #4289db;">
        <l-logo><a href="/" target="_blank">ICP 备案系统</a></l-logo>
    </div>
    <div style="background: #1d2b40;">
        <l-logo name="163yun">管理平台</l-logo>
    </div>
</u-linear-layout>
```

### 颜色扩展

``` html
<u-linear-layout direction="vertical">
    <div style="background: #343f4f;">
        <l-logo name="163yun" color="2color"></l-logo>
    </div>
</u-linear-layout>
```
